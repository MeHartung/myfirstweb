Meet App - Project Details

Project Description

The "Meet App" project was developed to display upcoming events with filtering options by city. The app also includes charts that show the number of events by city and event genres. The project was developed using the Google Calendar API and showcases skills in React, PWA, and data visualization with the Recharts library.

Project Objective

The goal of the project was to create an event-displaying app with filtering capabilities that works as a Progressive Web App (PWA) and displays data through charts.

Technologies Used

  • React.js
  • Recharts
  • Google Calendar API
  • Progressive Web App (PWA)
  • GitHub Pages for deployment
